
# Plane Seat Management System


This project is a Java-based application developed to manage and optimize the allocation of seats on an airplane, adhering to specific coursework specifications. As part of a university coursework, this system aims to simulate the seat reservation process for passengers on a commercial flight, while adhering to various constraints and requirement

## Installation

    1. Clone the repository to your local machine using the following command:
    git clone https://github.com/anjaya02/PlaneManagementSystem

    2. Open the project folder in intellij idea

 

## Usage

    1. Select the PlaneManagement.java class which contains the main method. 

    2. Click on run button

    2. You can now interact with the project in the intellij idea
